# Corpus brut

Deposer ici les documents du corpus avant indexation.

Pour le rendu, noter pour chaque fichier :

- nom du fichier ;
- source officielle ;
- date de telechargement ;
- perimetre du document.

Exemple :

```text
mini_rag_duree_travail.pdf
Source : Legifrance
Date de telechargement : YYYY-MM-DD
Perimetre : corpus de demonstration RAG
```
